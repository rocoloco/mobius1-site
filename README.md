# Mobius 1 — Company Landing Page

Static landing page for [mobius1.io](https://mobius1.io). Deployed via Cloudflare Pages.

Mobius 1 LLC builds **Yave**, an AI-powered WhatsApp platform for real estate agencies in Spain.

## Deploy

Connected to Cloudflare Pages. Pushes to `main` auto-deploy.

## Local Preview

```bash
npx serve . -p 3333
```
